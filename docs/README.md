# Documentation Index

This `docs/` directory gathers the latest deployment, automation and project guidance files for **TryOnMe**. Start with [deployment_overview.md](deployment_overview.md) for a concise summary.

- **despliegue.md** – Spanish version of the deployment guide.
- **deployment_review.md** – Final checklist before going live.
- **deploy_vercel_make.md** – Steps to trigger Vercel deployments from Make.

## Automation

- **flujo_automatizado_semanal.md** – Weekly schedule of key agents.
