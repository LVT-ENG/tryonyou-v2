# Make Scenarios

This folder contains automation scenarios for [Make](https://www.make.com/). Import the JSON files from `scenarios/` into your Make account to replicate each automation.

## Available scenarios

- **make_upload_scenario.json** – Envía datos de productos al endpoint `https://tryonyou.vercel.app/api/upload`.
- **domain_connect_flow.json** – Automatiza la conexión de dominio en Shopify utilizando los endpoints `/connect_domain`, `/update_dns`, `/set_primary_domain` y `/verify_dns`.
- **invite_auto_accept_bot.json** – Acepta invitaciones por email de forma automática.

Each scenario can be customized in Make to fit your workflow.
