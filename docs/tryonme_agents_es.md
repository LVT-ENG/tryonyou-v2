# Tryonme Agents

Este documento resume los distintos agentes virtuales propuestos para TRYONME. Cada uno tiene un rol y una serie de funciones específicas.

## Lista de agentes

1. **PMV – Project Manager Virtual**
   - Coordina y supervisa el sistema completo.
   - Centraliza tareas y promueve acciones prioritarias.
   - Activa otros agentes según necesidades del proyecto.
   - Monitorea estado y progreso en plataformas colaborativas.

2. **Content Pro – Redactor Web & Marca**
   - Genera y traduce contenido web.
   - Redacta textos para homepage, fichas de producto y blogs.
   - Adapta el tono de marca.
   - Traduce automáticamente a EN, FR, ES.

3. **Ficha Técnica Master – Producción & CMS**
   - Crea fichas técnicas y prepara datos para fábrica y CMS.
   - Incluye medidas, tejidos, cuidados y mockups.
   - Entrega versión para fábrica y versión para Shopify.
   - Automatiza subida de fichas técnicas.

4. **Proveedor Tracker – Control de Fábricas**
   - Gestiona presupuestos y relaciones con proveedores.
   - Redacta emails de presentación profesional.
   - Adjunta archivos listos para producción.
   - Compara costes, calidad y tiempos de entrega.

5. **RRSS Automator – Redes Sociales Inteligentes**
   - Publica y gestiona contenidos en redes sociales.
   - Genera textos e imágenes para posts, reels e historias.
   - Automatiza publicaciones con Zapier, Make o Meta API.
   - Adapta contenido por canal (Instagram, TikTok, etc.).

6. **Tester UX Web – Usuario Fantasma**
   - Simula navegación web para detectar errores.
   - Evalúa experiencia de usuario.
   - Sugiere mejoras en interfaz y conversión.
   - Reporta cuellos de botella en el funnel.

7. **Factory Master – Producción & Archivos**
   - Gestiona todo el material que va a fábrica.
   - Prepara paquetes con imágenes y fichas.
   - Valida formatos de impresión.
   - Supervisa muestras y versiones finales.

8. **Mockup Artist – Visualizador de Chaquetas**
   - Crea mockups realistas con IA.
   - Convierte diseños planos en imágenes 3D.
   - Entrega visuales para web, catálogo o vídeo.
   - Permite personalización de fondo y pose.

9. **Checkout UX Master – Optimización del Checkout**
   - Reduce fricción en el proceso de compra.
   - Optimiza CTAs y formularios.
   - Propone interfaz más clara y emocionalmente efectiva.
   - Analiza tasa de abandono y sugiere mejoras.

10. **Look Curator – Curador Estético + Comercial**
    - Selecciona y prioriza productos estrella.
    - Analiza impacto visual y atractivo comercial.
    - Ordena productos en portada, pasarela o colecciones.
    - Integra criterios de moda y ventas.

11. **Fit-AI Assistant – Asistente de Tallas Inteligente**
    - Recomienda prendas según el cuerpo del usuario.
    - Analiza medidas y preferencias previas.
    - Compara con base de datos de patrones textiles.
    - Sugiere talla perfecta y ajustes recomendados.

12. **Brand Guardian – Guardián de la Identidad**
    - Valida estética y tono de marca.
    - Revisa que todo el contenido sea coherente.
    - Detecta desviaciones visuales o verbales.
    - Aprueba publicaciones o diseños antes de ser visibles.

13. **HR Supervisor – Recomendador de Agentes**
    - Optimiza el uso del equipo GPT.
    - Sugiere el agente ideal para cada tarea.
    - Evalúa rendimiento y tiempo de respuesta.
    - Mejora flujos y uso eficiente del sistema.
