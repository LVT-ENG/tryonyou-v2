# TRYONYOU–ABVETOS–ULTRA–PLUS–ULTIMATUM

MVP público con **Vite 7.1.2 + React**:
- Hero + **8 Super-Claims**
- Botón **Probar** (modal avatar placeholder)
- CTA **Pagar con ABVET** (stub → #/abvet-payment)
- Footer con /patent y contacto

## Scripts
- `npm ci && npm run dev` – local
- `npm run build` – producción
- `npm run preview` – previsualizar build
- `npm run deploy` – Vercel (requiere VERCEL_TOKEN)

## Requisitos
- Definir secrets en GitHub/Vercel: `VERCEL_*`, `TELEGRAM_*`.
- Conectar dominio: **www.tryonyou.com**.
