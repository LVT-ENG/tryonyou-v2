# CRAWLER SNAKE

Este directorio documenta la versión inicial del sistema modular **CRAWLER SNAKE** encargado de aplicar pintura o recubrimiento mediante desplazamiento horizontal y vertical controlado.

## Contenido
- `arquitectura_general.md`: descripción de los módulos mecánicos, rieles y sistema de anclaje.
- `software_embarcado.md`: lógica de control, sensores y protocolos de seguridad.
- `piezas_impresas_3d.md`: listado y especificaciones de las piezas fabricadas en impresoras 3D.

Toda la información técnica sigue las pautas solicitadas para la documentación de patente.
