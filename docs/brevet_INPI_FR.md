# Brevet INPI - CRAWLER SNAKE

**Fig.1** Vista frontal del módulo sobre riel.
**Fig.2** Detalle del sistema de anclaje modular.
**Fig.3** Aplicación vertical con rodillos de adherencia.

El dispositivo CRAWLER SNAKE permite la aplicación automatizada de pintura o recubrimientos en superficies planas mediante desplazamiento controlado por motores y guiado sobre rieles.

El sistema incluye sensores que corrigen la trayectoria y un protocolo de seguridad que detiene el avance en caso de pérdida de adherencia.
