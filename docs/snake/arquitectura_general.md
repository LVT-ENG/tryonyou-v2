# Arquitectura General

El sistema CRAWLER SNAKE consta de módulos conectables que se desplazan sobre rieles horizontales y verticales.

## Módulos Principales
1. **Unidad de Tracción**: motor con engranajes para avance controlado.
2. **Sistema de Rieles**: guías modulares que permiten el desplazamiento vertical y horizontal.
3. **Anclaje Modular**: soportes ajustables para fijar el sistema a distintas superficies.

El diseño favorece el mantenimiento rápido y la sustitución de componentes.
