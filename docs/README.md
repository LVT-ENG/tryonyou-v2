# Documentation Index

This `docs/` directory gathers the latest deployment, automation and project guidance files for **TryOnMe**. The canonical reference is [deployment_overview.md](deployment_overview.md).

## Deployment
- **deployment_overview.md** – Consolidated deployment steps.
- **despliegue.md** – Spanish version of the deployment guide.
- **repository_structure.md** – Quick guide to the key folders in this repo.

## Automation
- **automa_flows.md** – Documentation for the Automa browser extension flows.
- **automa_make_flow.md** – Combining Automa with Make for continuous operations.
- **flujo_automatizado_semanal.md** – Weekly schedule of key agents.
- Use `automa_flows/site_health_email.automa.json` with the Automa extension to receive uptime alerts via email.

## Other references
- **shopify_domain_setup.md** – Connect your custom domain to Shopify.
- **financial_infrastructure.md** – Overview of payment and reward handling.
- **implementation_plan.md** – High‑level roadmap of features and data flows.

Scripts like `seo_audit.py` (SEO validation), `site_health_check.py` (email alerts) and `auto_update.py` (dataset and product sync) are in the `scripts/` directory. See each document for more details.
