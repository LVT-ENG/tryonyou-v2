# TryonMe Agents

This directory contains agent modules used across the project. The primary active agents are:

- **Fit-AI Assistant** – size recommendation assistant.
- **Look Curator** – arranges products and highlights attractive looks.
- **BrandGuardian** – validates content for brand consistency.
- **Checkout UX Master** – checks cart data to keep the purchase flow smooth.

Each agent exposes a function to be consumed by the main application or workflows.

Additional scripts under this directory provide utilities for deployment and testing.
